> Make sure you're on v2.4.0 or later — earlier versions had a gap in the damage hook for The Deer's lunge attack specifically. Update to the latest release.

> Some items in the game's newest structures (Bank, Abandoned Shelter) require the v2.3.0+ map index. Update the script and re-enable the feature to cover the full map.

**❌ Script crashes on mobile immediately after execution**
> Lower your Roblox graphics settings to Level 1 or 2 before injecting. High graphics combined with executor memory overhead is the most common cause of mobile crashes.

**❌ Teleport sends me underground or off-map**
> This is caused by stale teleport coordinates from a map update. Ensure you're on the latest script version. If the issue persists, use the "Safe TP" toggle in the Teleport tab, which adds a height offset before landing.

**❌ ESP stops showing entities after a few minutes**
> This is caused by Roblox's server-side entity streaming refreshing tracked objects. Toggle ESP off and back on in the GUI to force a re-scan of all current entities.
